# Dr. Swatantar Kumar - Research Portfolio

A modern, responsive personal portfolio website showcasing research contributions in **Microbial Ecology** and **Environmental Biogeochemistry**.

---

## Overview

This portfolio presents Dr. Swatantar Kumar's academic journey, research contributions, and professional achievements in the field of environmental microbiology and biogeochemistry. The website features a clean, professional design with smooth animations and an intuitive user experience.

### Key Sections

- **Hero Section** - Introduction with professional background and multi-layer animated visuals
- **About Section** - Academic background, education, and research interests
- **Experience Timeline** - Interactive drag-and-scroll timeline of positions across research institutions in India, Germany, and USA
- **Research & Publications** - Scientific contributions organized by research domains (Nitrogen Cycling, Rhizosphere Biogeochemistry, Environmental Microbiology)
- **Professional Achievements** - Patents, peer review activities, invited talks, and field expeditions
- **Contact Section** - Multiple channels for collaboration and communication

---

## Tech Stack

- **Framework**: Next.js 14
- **Styling**: Tailwind CSS
- **Animations**: Framer Motion
- **Icons**: Lucide React
- **Deployment**: GitHub Pages

---

## Development

```bash
# Install dependencies
npm install

# Run development server
npm run dev

# Build for production
npm run build

# Export static site
npm run export
```

---

## Deployment

The site is configured for automatic deployment to GitHub Pages via GitHub Actions. Push to the `main` branch to trigger the deployment workflow.
